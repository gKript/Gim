Detailed Description

The Gim's file object class is useful in order to use the files like objects fastly and easily. _gim_file is a type defined as this class. In this example we create a backup of our precious /etc/X11/xorg.conf
.

#include <gim/gim.h>
int main( int argc , char **argv ) {
gim_obj * gim = new gim_obj;
_gim_file source( "/etc/X11/xorg.conf" );
_gim_file destination( "/etc/X11/xorg.conf.backup" , __GIM_FILE_POINTER , __GIM_WRITE );
destination.flush( source.fm() , source.size() );
source.close();
destination.close();
delete gim;
return 0;
}
